Implementace

app/model obsahuje php skripty pro práci s jednotlivými tabulkami databáze a skript pro ověření přístupu
app/presenters/newPresenter.php php skript, který obsahuje funkce pro vytvářeni formulářů pro:
  • vytvoření zaměstnance
  • vytvoření oddělení
  • vytvoření zakázky
  • vytvoření zadavatele
  • ...
  • dále obsahuje funkce pro validaci formulářu a vložení dat do databáze
app/presenters/homepagePresenter.php php skript, který obsahuje funkce pro zobrazení informací o:
  • změstnancích
  • zakázkách
  • odděleních
  • zadavatelích
  • ...

Instalace

  • Ke spuštění je potřeba nainstalovat Nette Framework 2.0.7. Ke stažení zde.
  • Dále doplňky k nette. Ke stažení zde a zde.
  • Archiv s aplikací rozbalit a vložit do NetteFramework/ na serveru.
  • Softwarové požadavky:
    • PHP verze 5.3 a vyšší
    • MySql verze 5.5 a vyšší
  • Archviv obsahuje i sql script pro vytvoření databáze a naplnění testovacími daty - pro instalaci stačí zkopírovat obsah souboru v phpMyAdmin (záložka SQL)
